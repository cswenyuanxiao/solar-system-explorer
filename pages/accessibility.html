Tab Navigate through interactive elements
Arrow Keys Navigate vertically and horizontally
Enter Activate buttons and links
Space Activate buttons and checkboxes
Escape Close modals and dropdowns
Ctrl + H Toggle high contrast mode
Ctrl + V Toggle voice control
Ctrl + + Zoom in
Ctrl + - Zoom out
Ctrl + 0 Reset zoom

Voice Commands

Enable voice control and use these commands to navigate the application:

Navigation Commands

  • "Navigate to [planet name]" - Go to a specific planet
  • "Go home" - Return to main page
  • "Go back" - Navigate to previous page
  • "Search [term]" - Search for content

Control Commands

  • "Zoom in" - Increase zoom level
  • "Zoom out" - Decrease zoom level
  • "High contrast" - Toggle high contrast mode
  • "Help" - Show available commands
